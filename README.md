
# BookLoop

Plataforma web para comprar, rentar y opinar sobre libros.

---

## Introducción / Contexto

* **Descripción del problema que se busca resolver:**
  Actualmente muchas personas pasan la mayor parte del tiempo en dispositivos electrónicos consumiendo contenido rápido y dejando de lado la lectura tradicional. Además, no todos pueden comprar libros nuevos constantemente, y muchas veces solo quieren leer un libro una vez sin necesidad de conservarlo.

* **Justificación:**
  Este proyecto es relevante porque busca fomentar la lectura ofreciendo una alternativa accesible mediante la compra y renta de libros. Tiene impacto social al incentivar el hábito de leer, impacto académico porque aplica los conocimientos adquiridos en programación y bases de datos, y también impacto tecnológico al integrar múltiples herramientas modernas.

* **Breve descripción del dominio / temática del proyecto integrador:**
  Se trata de una tienda virtual de libros que combina funcionalidades de e-commerce (como carrito de compras y lista de deseos) con características de red social (comentarios y opiniones por libro), siendo una mezcla entre una tienda como Amazon y un foro tipo Twitter enfocado en lectura.

---

## Objetivos

### Objetivo General

Desarrollar una aplicación web para la compra, renta y discusión de libros utilizando Python y tecnologías web modernas.

### Objetivos Específicos

* [OE1] Diseñar e implementar un sistema de registro e inicio de sesión de usuarios.
* [OE2] Crear un catálogo de libros con opción de compra y renta.
* [OE3] Implementar un carrito de compras y lista de productos deseados.
* [OE4] Desarrollar un sistema de comentarios y opiniones por cada libro.
* [OE5] Integrar una base de datos que permita almacenar usuarios, libros, pedidos y comentarios.

---

## Alcance del Proyecto (Scope)

### ¿Qué se va a desarrollar?

* Sistema de autenticación de usuarios (registro e inicio de sesión).
* Catálogo dinámico de libros disponibles para compra y renta.
* Carrito de compras funcional.
* Lista de deseos (wishlist).
* Sistema de comentarios tipo foro en cada libro.
* Conexión a base de datos mediante Python.
* Interfaz web desarrollada con HTML, CSS y JavaScript.

### ¿Qué NO se va a desarrollar en esta versión (fuera de alcance)?

* Aplicación móvil nativa.
* Pasarela de pagos real (solo simulación de compra).
* Sistema avanzado de recomendaciones con inteligencia artificial.
* Envíos físicos o logística real.
* Chat en tiempo real entre usuarios.

---

## Tecnologías y Herramientas (Tech Stack)

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python, Java
* **Base de datos:** MySQL / SQL Server
* **Otras herramientas:** Git, GitHub, Streamlit, React, SpringBoot
